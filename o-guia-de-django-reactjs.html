O guia de Django + React.js !

Renan

Ter 14 julho 2015
Categoria: Python
Tags: django, react
 

Depois de muito tempo, "fugindo" do jQuery, de alguns anos para cá, diversas outras bibliotecas JavaScript foram aparecendo, pela explosão do Javascript no lado do serve-side (node.js), a padronização da própria linguagem Javascript e a procura por aplicações sempre mais ágeis com alto nível de performance. Você já deve ter ouvido falar de algumas dessas bibliotecas AngularJS, Backbone, Ember.js, entre muitas coisas. Uma que me chamou muito a atenção é o React.js, reconheço vários motivos para esse interesse, incluindo seu ecossistema, a comunidade, sua sintaxe, o principal é ele trazer a questão de performance para o front-end, apesar de que há projetos usando essa lib no server-side também, a forma na qual o React foi feito e o conceito que ele usa, trás questões parar melhor tratamento do DOM em Javascript.

Um resumo rápido do que é o React é uma biblioteca Javascript para criação de Interfaces, é mantida pela Facebook, vinda do Instagram, ele usa o conceito de componentes (component driven development), onde partes importantes do sistema/interface são divididos em um ou mais componentes com funções claras, não vou me estender nisso, logo a baixo alguns links para estudo do React:

  • https://facebook.github.io/react/
  • https://github.com/facebook/react
  • http://willianjusten.com.br/comecando-com-react/

A intenção deste primeiro post, é falar da junção de Django, REST e React, em vários tutoriais, pela internet é mostrado de uma maneira simples, apenas adicionar uma ou duas tag's <script>, porém depois de algumas semanas usando dessa forma, percebe-se que para alguns não é uma forma que irá atender 100%, pelo motivo que algumas lib's do react estão em node.js, inclusive, a grande maioria dos exemplos até agora são react+node.js, então resolvi juntar isso, para poder usar todo o ecossistema que está envolta a biblioteca (Por exemplo: o Flux, JSX, entre outros) de início, segue a lista de ferramentas necessárias para acompanhar esta série de posts:

  • Python: É necesario ter o python instalado em sua máquina
  • Django: Framework Web escrito em Python
  • Django REST Framework: API REST usando Djando
  • Node.js & NPM:
    • React
    • JSX-Loader
    • Gulp
    • Webpack
    • Browserify
    • Flux

Nos próximos posts iremos abordar cada uma das ferramentas, para você não achar que é só isso, todo o código que irá envolver essa série já está 42% pronto e disponível no Github do autor:

  • https://github.com/RenanPalmeira/django-react
  • Django+React+Webpack
  • Djando+React+Gulp
  • Django+React+Grunt: Em andamento
